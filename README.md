# Python Banking System

A prototype of a banking system developed in Python as a personal project for learning and experimentation.

## Features

- Create bank accounts
- User login
- Deposit money
- Withdraw money
- Check account balance

## Technologies

- Python 3

## Future Improvements

- Better project architecture
- Graphical user interface (GUI)
- Database integration
- Password encryption
- Transaction history
- Better account management

## Learning Goals

This project was created to practise:

- Object-Oriented Programming (OOP)
- File handling
- Project organisation
- Git & GitHub workflow
- Writing maintainable Python code
